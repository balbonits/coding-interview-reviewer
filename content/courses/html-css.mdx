The frontend foundation track. Semantic HTML and accessibility are non-negotiable for senior front-end roles — bad ARIA shows up immediately in code reviews.

CSS coverage favors *modern* (`:has()`, container queries, cascade layers) — older techniques like floats and table layouts deliberately get no time.
