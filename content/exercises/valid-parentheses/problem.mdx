---
title: "Valid Parentheses"
---

Given a string `s` containing only the characters `()[]{}`, return `true` if every opening bracket is closed by the same type of bracket and brackets are closed in the correct order.

## Examples

```js
isValid("()")           // → true
isValid("()[]{}")       // → true
isValid("(]")           // → false
isValid("([)]")         // → false   (interleaved)
isValid("{[()()]}")     // → true
isValid("")             // → true
isValid("(")            // → false   (never closed)
```

## Hint

This is the canonical stack problem. Iterate the string. Push opening brackets onto a stack. On a closing bracket, pop the top of the stack and check it's the *matching* opener. If the stack is empty when you try to pop, the string is invalid. After the loop, the stack must be empty.

A `Map` from closer → opener (`')' → '('`, `']' → '['`, `'}' → '{'`) keeps the matching logic readable.
